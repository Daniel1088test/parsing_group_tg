Telegram Authentication Guide

How to Authenticate Telegram Sessions

There are two ways to authenticate your Telegram sessions:

  1. Via the website interface (recommended)
  2. Through the Telegram bot directly

Method 1: Website Authentication

This is the easiest way to authenticate a session:

  1. Go to the Sessions List page
  2. For any session that shows "Needs Auth", click the "Authorize" button
  3. Follow the instructions on the screen to connect with the Telegram bot
  4. Enter your phone number and the verification code sent by Telegram
  5. Once authentication is complete, return to the website

Method 2: Direct Bot Authentication

You can also authenticate directly through our Telegram bot:

  1. Add our bot @Channels_hunt_bot on Telegram
  2. Send the command /authorize to the bot
  3. Share your phone number when prompted
  4. Enter the verification code sent by Telegram
  5. The bot will confirm when authentication is successful
Important Notes:
  • You must use a regular Telegram user account, not a bot account
  • You need access to the phone number you're authenticating
  • If you have two-factor authentication enabled, you may need to provide your password
  • One phone number can only be associated with one session
