Telegram Authentication Guide

This guide will help you authorize your Telegram account for channel parsing. No terminal commands are needed - everything can be done through the bot or web interface.

How to Authenticate Telegram Sessions

Method 1: Via Website (Recommended)

This is the easiest way to authenticate a session:

  1. Go to the Sessions page
  2. Click the "Authorize" button next to the session that needs authentication
  3. Open the Telegram link provided on the next screen
  4. Our bot will guide you through the authentication process
  5. Enter your phone number and verification code when prompted
  6. If your account has two-factor authentication, enter your password when asked
Advantage: This method lets you authorize directly from the web interface with just a few clicks.
Method 2: Directly Through the Bot

You can also authenticate by talking directly to our Telegram bot:

  1. Open our bot @Channels_hunt_bot on Telegram
  2. Send the command /authorize to the bot
  3. Share your phone number when prompted
  4. Enter the verification code sent by Telegram
  5. If your account has two-factor authentication, enter your password when prompted
Advantage: Convenient if you're already using Telegram and don't want to switch between applications.
Important Notes
  • You must use a regular Telegram user account (not a bot account)
  • The phone number must have access to Telegram
  • Two-factor authentication (2FA) is fully supported through both methods
  • If you encounter any issues, try refreshing the page or restarting the process
